Getting Started

#### 1. Clone the Repo

```bash
git clone https://github.com/Vraj-15/securevaultx.git
cd securevaultx
```

#### 2. Backend Setup

```bash
cd backend
npm install
node server.js
```

#### 3. Frontend Setup

```bash
cd frontend
npm install
npm start
```

#### 4. Database Setup

- Ensure PostgreSQL is installed and running
- Configure DB credentials in `backend/config.js`
- Run migration scripts or manually create tables
